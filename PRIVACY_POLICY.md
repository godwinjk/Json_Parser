# Privacy Policy

**Plugin**: Advanced Json Studio
**Developer**: Godwin Joseph Kurinjikattu
**Last updated**: April 2026

### Overview
Advanced Json Studio is an IntelliJ plugin built to help developers work with JSON efficiently. We are committed to being transparent about how the plugin works and what happens to your data. The short version: your JSON stays on your machine.

### Anonymous Usage Analytics
We collect anonymous usage data to understand how the plugin is used and to improve the experience. This includes:

- Which features and actions are triggered
- How often certain functions are used
- Plugin version, IDE version, and operating system type

This data is fully anonymous and cannot be linked back to you or your machine.

### How to Opt Out
Anonymous data collection can be disabled at any time from inside the plugin:

> Settings Icon → Privacy → Enable Analytics


### AI Features & Your Data
Advanced Json Studio includes two AI-powered features. Each is handled differently and transparently:
#### AI Code Generation (from JSON Schema)

This feature generates code using your JSON Schema, not your actual JSON data
Your raw JSON content is never sent to any server for this feature
Only the schema structure is used, which contains no personal or sensitive information

#### AI JSON Repair
- This feature requires sending your JSON to an AI model in order to analyse and repair it
- Your JSON is processed in real time and is not stored, logged, or retained after the response is returned
- Your JSON data is never used to train AI or machine learning models
- Your JSON data is never sold or shared with any third party


### What We Do NOT Collect
We do not collect, store, or process:

- Your name, email, or any personal identifiers
- Your IP address or device fingerprint
- Any contents of your clipboard or editor beyond what you explicitly invoke a feature on
- Any personally identifiable information (PII) of any kind


### Your JSON & Local Storage
Your JSON data belongs to you and stays on your machine:

- JSON content you work with is stored in IntelliJ IDEA's local cache on your device
- This cache is not synced to any cloud service
- This cache is not uploaded or stored on our servers
- We have no access to your local cache files at any time


### Third Parties & AI Training

- Your data is never sold or shared with third parties
- Your data is never used to train AI or machine learning models
- We do not use third-party advertising networks or tracking services


### Changes to This Policy
We may update this policy as the plugin evolves. Any changes will be published here with an updated date at the top. Significant changes will be communicated via the plugin's changelog.
